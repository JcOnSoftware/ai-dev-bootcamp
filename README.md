# ai-dev-bootcamp (workspace)

Workspace de trabajo del proyecto **ai-dev-bootcamp**: una CLI interactiva estilo `rustlings` para que devs senior aprendan a usar tools de AI (Claude API, prompt caching, tool use, RAG, agents, MCP) con ejercicios progresivos y tests automáticos.

## Estructura

- **`code/`** — Código fuente (monorepo Bun workspaces)
- **`docs/`** — Documentación: plan, arquitectura, guía de contribución, ADRs
- **`notes/`** — Notas de trabajo, ideas sueltas, decisiones pendientes
- **`research/`** — Investigación: análisis de rustlings, competencia, benchmarks

## Punto de partida

Ver [`docs/PLAN.md`](./docs/PLAN.md) — plan completo con contexto, arquitectura, stack, tracks y roadmap.

## Enfoque actual

Milestone 1 — MVP. Empezamos por el **harness del runner + primer ejercicio** para validar el contrato de tests antes de escalar al CLI completo.
