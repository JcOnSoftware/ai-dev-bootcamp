## Packages

- **`@aidev/runner`** — Harness que ejecuta código del user e intercepta llamadas al Anthropic SDK para que los tests puedan hacer assertions sobre request + response.
- **`@aidev/exercises`** — Contenido: ejercicios versionados organizados por tracks.
